# 영도 저장소 — 릴리즈 노트

## v0.1.0 (첫 릴리즈)

치지직 영상도네(영도)용 클립 저장소. 내가 쏜 영도를 자동으로 모으고, 폴더·태그로 정리하고, 후원할 때 바로 검색·복사한다.

### 주요 기능
- **영도 자동 수집** — 치지직 후원내역에서 영상 후원(YouTube / 치지직 클립)을 자동으로 불러와 저장
- **수동 추가** — URL로 직접 추가, 시작/종료 구간 지정 가능
- **편집/관리** — 제목·URL·구간 인라인 수정, 즐겨찾기, 삭제(재동기화해도 다시 안 살아남)
- **폴더 · 태그** — 중첩 폴더(폴더 안 폴더)와 태그로 분류, 검색·필터
- **전용 관리 페이지** — 폴더 트리, 하위 폴더 생성, 체크박스 일괄 이동, 영도 추가
- **후원 패널** — 치지직 후원하기 창 옆에 영도 목록을 띄워 영도 중 바로 검색·복사 (반응형)
- **테마 연동** — 치지직 라이트/다크 테마를 자동으로 따라감
- **구간 복사** — 유튜브는 복사 시 `start`/`end` 파라미터가 URL에 반영됨

### 알려진 제약
- 후원내역 API는 **치지직 로그인 상태**에서만 동작 (본인 인증 쿠키 사용)
- 치지직은 영도 시 잘랐던 구간을 저장하지 않으므로, 구간은 이 확장에서 직접 입력해 보존
- 저장은 현재 **로컬(브라우저)** 기준. 기기 간 동기화는 추후 검토

### 데이터
- 모든 데이터는 `chrome.storage.local`에 저장되며 외부로 전송되지 않음
